Description

A generator that directly reads inspiring images, renders parameterized images, and adjusts its own designs to optimize an evaluation score.

Technical

I am creating randomly generated mutations of an impressionalist version of the provided images. My program creates circles throughout the canvas of random colors in hopes of matching the inspiration image. The mutation that is closest to the inspirational image is constantly shown on the right, only updated when a better match occurs.

Reflection

The most irritating part of this experiment was beginning this project. Initially, I struggled to understand the technical requirements as well as how to implement my own design. With help from my classmates and roommates, I was able to successfully implement everything I had wanted to.
