Experiment 1 - Shitpost Generator

Translated from cmpm-147-ex01-shitpost on glitch.com

Description

Experiment with and create a shitpost generator and move it to my portfolio page.

Technical

Represented key words and phrases using simple replacement tokens. A replacer function reads the template and replaces any token with a random replacement word or phrase.

Reflection

This project was super fun and simple. The hardest part has been trying to figure out how to integrate it into my website.
